# Дашборды для анализа компании Яндекс

Проект содержит 5 интерактивных дашбордов для визуального и контекстного анализа компании Яндекс на основе финансовых, ESG-данных, новостного фона и прогнозов аналитиков.

## Состав проекта

| Дашборд | Описание |
|---------|----------|
| **Финансовое здоровье** | Ключевые метрики, динамика показателей, LLM-комментарии, drill-down по кварталам |
| **ESG-аналитика** | Динамика ESG-метрик, матрица существенности, сравнение с отраслевыми бенчмарками и целями SDGs |
| **Интегрированный анализ** | Финансы, ESG, тональность новостей и курс акций на одном экране, корреляция и таймлайн событий |
| **Мониторинг рисков** | Тепловая карта рисков, алерты при изменении профилей, автоматическая генерация PDF-отчёта |
| **Прогнозы vs Факт** | Сравнение консенсус-прогнозов с фактом, earnings surprise, LLM-анализ расхождений |

## Технологии

- **Streamlit** — фреймворк для дашбордов
- **Plotly** — интерактивные графики
- **Ollama** — локальная LLM (модель qwen2.5:3b)
- **Pandas** — обработка данных
- **ReportLab** — генерация PDF

## Установка и запуск

### 1. Клонировать репозиторий
```bash
git clone git@github.com:vikvlr/dashboard.git
cd dashboard
```

### 2. Создать виртуальное окружение и активировать
```bash
python3 -m venv venv
source venv/bin/activate  # macOS
venv\Scripts\activate     # Windows
```

### 3. Установить зависимости
```bash
pip install -r requirements.txt
```

### 4. Установить Ollama и скачать модель
```bash
# Установить Ollama с https://ollama.com
ollama pull qwen2.5:3b
```

### 5. Запустить дашборды
```bash
streamlit run app.py
```

После запуска откроется главная страница с навигацией по всем дашбордам.
